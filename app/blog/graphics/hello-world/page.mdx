## MDXのパワーを体験する

これは普通のMarkdownテキストです。箇条書きも簡単に書けます。

- Next.js
- React Three Fiber
- MDX

そして、ここからが本番。記事の中に**3Dシーンを直接埋め込みます**。

    <Box position={[0, 0, 0]} />

### 数式のテスト（Client Side）

数式を表示するには、`$` で囲みます。
MDX内でバックスラッシュを使うときは、**2回重ねて (`\\`) エスケープする**点に注意してください。

インライン数式: $E = mc^2$

ディスプレイ数式（分数の例）:

<div className="my-4 overflow-x-auto">
{`$$
\\frac{1}{2\\pi i} \\oint_\\gamma f(z) dz
$$`}
</div>

### コードブロックのテスト

シンタックスハイライトも効いているはずです。

```tsx
// Reactコンポーネントの例
import { useState } from 'react'

export default function Counter() {
  const [count, setCount] = useState(0)
  return <button onClick={() => setCount(c => c + 1)}>{count}</button>
}
```
