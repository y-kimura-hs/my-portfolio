## このブログの機能紹介

この記事は，このサイトの機能テストも兼ねています．
通常の Markdown 記法だけでなく，WebGL や数式などのコンテンツをシームレスに埋め込めるようにしました．

### WebGLデモの埋め込み

記事の中に React Three Fiber で作られた 3D シーンを直接配置できます．
以下のボックスは画像ではなく，実際に動いている 3D オブジェクトです．

<div className="w-full h-[400px] my-8 rounded-xl overflow-hidden border border-border bg-black/50 relative">
  <Scene>
    <Box position={[0, 0, 0]} />
  </Scene>
</div>

↑ 上のボックスは，マウスドラッグで回転させたり，クリックで大きくしたりできます．
記事を読みながら実際の挙動を確認できるので，技術ブログを書くのに便利です．

### シンタックスハイライト

コードブロックのシンタックスハイライトもできるようにしました．

```tsx
// Reactコンポーネントの例
import { useState } from 'react'

export default function Counter() {
  const [count, setCount] = useState(0)
  return <button onClick={() => setCount(c => c + 1)}>{count}</button>
}
```
